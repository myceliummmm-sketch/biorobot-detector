# Споря - Community Manager Bot

Ироничный и слегка токсичный (но добрый) комьюнити-менеджер для Mycelium чата.

## Возможности

- Отвечает на сообщения в групповом чате с Gemini AI
- Ироничная личность с легким сарказмом
- Тонко продвигает Mycelium Cards
- Реагирует на упоминания и ключевые слова
- Случайные ответы для поддержания активности

## Настройка

### 1. Создай бота в Telegram

1. Напиши @BotFather в Telegram
2. `/newbot` → назови его (например "Споря")
3. Скопируй токен

### 2. Получи Gemini API Key

1. Иди на https://makersuite.google.com/app/apikey
2. Создай API ключ (бесплатно)
3. Скопируй ключ

### 3. Добавь бота в чат

1. Добавь бота в групповой чат
2. Сделай его админом (чтобы видел все сообщения)

### 4. Настрой переменные окружения

```bash
cp .env.example .env
# Отредактируй .env файл
```

### 5. Запусти бота

```bash
pip install -r requirements.txt
python bot.py
```

## Деплой на Railway

1. Создай новый проект на Railway
2. Подключи GitHub репозиторий
3. Укажи root directory: `community_bot`
4. Добавь переменные окружения:
   - `COMMUNITY_BOT_TOKEN`
   - `GEMINI_API_KEY`
   - `RESPONSE_PROBABILITY` (опционально, по умолчанию 0.3)

## Команды бота

- `/start` - Приветствие
- `/status` - Статус бота
- `/clear` - Очистить память чата

## Настройка личности

Отредактируй `SYSTEM_PROMPT` в `config.py` чтобы изменить характер бота.
